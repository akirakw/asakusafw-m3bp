=========
|FEATURE|
=========

|FEATURE|\ は、Asakusa DSLを始めとするAsakusa Frameworkの開発基盤を利用して作成したバッチアプリケーションに対して、
|ENGINE|\ をその実行基盤として利用するための機能セットを提供します。

|FEATURE|\ の概要と利用方法は、 :doc:`user-guide` を参照してください。
